# Medivance - Smart Hospital Management System

Medivance is a premium, state-of-the-art smart hospital management system designed to make healthcare administration and access seamless, fast, and secure. 

## Key Features

* **Patient Dashboard**: View health records, book appointments, track history, and check active schedules.
* **Doctor Dashboard**: Manage incoming appointments, access patient records, and update prescriptions.
* **Admin Dashboard**: Comprehensive control center for department management, scheduling, staff roles, and analytics.
* **Smart Scheduling**: Seamlessly book and manage appointments with preferred specialists.
* **Emergency Portal**: Instant access to emergency support and protocols.

## Tech Stack

* **Frontend**: React, Vite, TypeScript, Tailwind CSS, Radix UI
* **State Management & Data Fetching**: TanStack Query (React Query)
* **Routing**: React Router DOM
* **Components**: shadcn/ui
* **Database/Backend**: Supabase Integration

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### Installation

1. Install the necessary dependencies:
   ```bash
   npm install
   ```

2. Start the development server with hot-reloading:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:8080/`.

### Building for Production

To create an optimized production build:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
